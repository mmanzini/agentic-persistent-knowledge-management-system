# Archive only

Dormant input. The agent does **not** read this folder during
`consolidate`. Use it for material that's worth keeping at hand for
human reference (e.g. a paper you've already read, an old export you
want to be able to grep) but that you don't want absorbed into the
wiki — either because it's already been consolidated elsewhere, or
because it's too noisy to be useful as a source.

To re-activate a folder, change `include_in_consolidation` to `true`
and run `consolidate`.
