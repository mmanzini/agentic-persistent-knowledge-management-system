# Web clippings

Articles, blog posts, and pages saved from the web for later
consolidation. Treat each `.md` file as one source. Images live next
to the clipping that references them and travel with that clipping
during consolidation. These are ephemeral inputs — once the agent has
absorbed them into the wiki, the originals are deleted from this
folder.
