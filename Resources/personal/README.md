# Personal

Static, hand-curated personal source material. The authoritative
about-me, writing-rules, and any other primary document the agent
should treat as a source-of-truth for who you are and how you work.

Files here are kept after consolidation — they're primary documents
in their own right. Edit them by hand when something fundamental
changes; let `consolidate` handle propagation into the wiki.

Each `.md` is a single source. Sources here are **prescriptive** —
when `consolidate` derives articles from them, it preserves the
original wording rather than paraphrasing. The voice is the point.
