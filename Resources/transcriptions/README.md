# Transcriptions

Meeting transcriptions, podcast transcripts, voice notes. Each `.md`
file is one source. Treat these as **primary documents** — they have
archival value of their own, so they are read and consolidated like
any other source, but the originals **stay** in this folder after
consolidation rather than being deleted. The wiki article in
`Intelligence/` is a derivative; the transcription here is the canon.
