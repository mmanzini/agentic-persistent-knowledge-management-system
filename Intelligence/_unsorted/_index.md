# _unsorted — quarantine

Articles land here when `consolidate` couldn't match them to any
existing bucket. This is a routing failure, not a factual problem.

## What to do

1. Review each article in this folder.
2. For each, decide:
   - **Create a new bucket.** Add a folder under `Intelligence/` with
     a `_master-index.md` describing its scope, register it in
     `Intelligence/index.md`, then move the article into the
     appropriate topic inside that bucket (creating the topic if
     needed) and update the topic's `_index.md`.
   - **Move into an existing bucket.** If on second look the article
     does fit somewhere, move it manually and update the relevant
     `_index.md`. Consider tightening the bucket's Scope paragraph
     so the next run routes correctly.
   - **Drop it.** Delete the article if it doesn't deserve a place
     in the wiki.
3. Append a one-line entry in `Intelligence/log.md` describing the
   resolution.
